== Description ==

<a href="http://workshop.rs/projects/coin-slider/">Coin Slider</a> 4 WP is Wordpress plugin for creating image gallery with unique transition effects of featured posts. You can choose between three types of transition effects: random, swirl and rain. Also, you can set width, height, delay time, number of posts etc.

You can find more informations about this plugin on <a href="http://workshop.rs/">www.workshop.rs</a>

== Installation ==

1. Upload coin-slider-4-wp folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. &lt;&#63;php include (ABSPATH . '/wp-content/plugins/coin-slider-4-wp/coinslider.php'); &#63;&#62;
4. For each post that you want to be featured you must have custom field (csImage by default) with full url to image and optionaly field (csText by default) with text for title bar. You can change default fields if you want to use some other.
